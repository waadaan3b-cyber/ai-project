# Book Rating Predictor (AI Project)

## Overview

This project implements a Book Rating Prediction system using Brain.js
neural networks in JavaScript.

The system analyzes a book review and predicts a rating from 1 to 5
stars.

## Technologies Used

-   HTML
-   CSS
-   JavaScript
-   Brain.js

## Features

-   Multi-class rating prediction (1--5 stars)
-   Confidence percentage display
-   Visual star representation
-   Clean modern UI design

## How It Works

1.  User enters a book review.
2.  Text features are extracted.
3.  Brain.js neural network processes the features.
4.  System outputs predicted rating and confidence score.
